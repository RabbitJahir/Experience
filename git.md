## Branches

> To make a new branch
> - `git checkout -b new-branch-name`
>
> Pressed on commit and push, but forgot to make a new branch?
> - `git checkout -b your-branch-name`
>
> To go back to main branch / origin.
> - `git checkout main`
>
> You might see a message saying your git is up to date with main or, Your branch is behind 'origin/main' by 9 commits, and can be fast-forwarded. (use "git pull" to update your local branch)
> - `git pull`
> this only pulls the updates on your device, no merging, dont worry!
